# DSA Java Problem Archive

A curated collection of Java-based Data Structures and Algorithms practice problems solved across basics, recursion, arrays, strings, OOP, and evaluation sets.

## Highlights

- Language: **Java**
- Total solved files: **139 Java solutions**
- Includes reusable bit-manipulation templates: [`src/BitTemplates.java`](src/BitTemplates.java)

## Bit Manipulation Quick Start

Use [`src/BitTemplates.java`](src/BitTemplates.java) for ready-to-use methods:

- Sum without `+`
- Power-of-two / power-of-four checks
- Set/clear/toggle k-th bit
- Single number / two single numbers (XOR)
- Range bitwise AND, reverse bits, subset generation
